### 游戏卡牌
- 简介：可以生成三国杀游戏的武将牌
- 测试平台：NewBing
- 关键词：搜索三国杀武将卡，请结合_______的特点,为他生成一张这样的卡牌。（横线处填写你想要，创造的人物的名称，如果这个人不存在则你可以让newbing根据他的特点自行进行生成。）
